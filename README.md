# Бот для прохождения тестирования hrbot

## Описание
Бот для прохождения тестирования.
Бот предназначен для автоматизации процесса прохождения тестов и получения результатов тестирования.


В ходе разработки особое внимание уделяется качеству кода, прозрачности процесса,
документированию решений и регулярной обратной связи через code review.

---

## Цель проекта
Цель проекта — разработка работоспособного бота для прохождения тестирования, а также
формирование практических навыков разработки программного обеспечения, включая:

- анализ требований и формирование логики работы бота
- работу с Git и GitHub (issues, ветки, pull requests)
- написание читаемого и поддерживаемого кода
- обработку ошибок и нестандартных ситуаций
- документирование функциональности и решений
- получение и внедрение обратной связи от наставника

## Формат работы
- Все задачи оформляются в виде **Issues**
- Разработка ведётся в отдельной ветке dev
- Изменения принимаются только через **Pull Request**
- Каждый Pull Request проходит review
- Прямые коммиты в основную ветку запрещены

## Ожидаемый результат
По итогам практики репозиторий должен содержать:
- работающего бота для прохождения тестирования
- исходный код с понятной структурой
- описание логики работы бота
- историю разработки и принятых технических решений
